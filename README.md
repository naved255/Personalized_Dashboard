# 📊 Personalized Dashboard

A lightweight, modern, and privacy-first personal productivity dashboard built with **React** and **Vite**. This application empowers users to manage tasks effectively against strict deadlines, monitor their overall productivity via an interactive progress speedometer, track live news, and customize their interface environment. All data is securely retained right on the user's local device.

## ✨ Key Features

- **🎯 Task & Deadline Tracker:** Easily create tasks with designated deadlines. Track status dynamically as tasks are marked either completed or failed.
- **🏎️ Productivity Speedometer:** Visually monitor your execution metrics through an interactive gauge reflecting the real-time percentage of completed tasks.
- **🌓 Adaptive Theme Engine:** Toggle smoothly between fully custom Light and Dark modes via the dedicated settings panel.
- **📰 Integrated News Hub:** Stay updated with real-time global headlines fetched dynamically from an external news API source.
- **💾 Privacy-First Architecture:** Zero backend or tracking servers. 100% of user data, task history, and UI themes are retained locally on the user's machine via `localStorage`.
- **⚙️ Advanced Settings:** Keep the workspace clean by purging historical logs or permanently deleting completed tasks.

## 🛠️ Tech Stack

- **Frontend Library:** React (Functional Components & Hooks)
- **Build Tool:** Vite (for near-instantaneous Hot Module Replacement)
- **Styling:** CSS3 / Vanilla CSS Variables (Custom-themed)
- **Data Persistence:** Web Storage API (`localStorage`)
- **Linting:** ESLint (Production-ready configuration)
